# 子模块

子模块允许您将一个Git存储库保存为另一个Git存储库的子目录。
这允许您将另一个存储库克隆到您的项目中，并保持您的提交是分开的。

## 添加子模块

使用 `git submodule add` 命令和您想要开始跟踪的项目的绝对或相对URL

```bash
git submodule add https://github.com/chaconinc/DbConnector
```
