# 安装 MAVLink 工具链

- [Pymalllink](https://pypi.org/project/pymavlink/)
- [Installing MAVLink Toolchain](https://mavlink.io/en/getting_started/installation.html)

```sh
# Install Python 3.7+
sudo apt install python3 python3-pip

# Clone the official mavlink repo
git clone https://github.com/mavlink/mavlink.git --recursive
cd mavlink

# Install the required packages
python3 -m pip install -r pymavlink/requirements.txt

# Verify installation
python3 -m pymavlink.tools.mavgen -h
```
