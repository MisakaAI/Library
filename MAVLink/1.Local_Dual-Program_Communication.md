# 本地双程序通信

用 UDP 在本机通信

- 程序 A：发送 MAVLink 消息
- 程序 B：接收并解析

## 依赖

Python + MAVLink 官方库

```sh
pip install pymavlink
```

## 示例（本地通信）

**1. 接收端（先运行）**

```py
# receive.py
from pymavlink import mavutil

# 监听本地 UDP 端口
master = mavutil.mavlink_connection('udp:127.0.0.1:14550')

print("Waiting for messages...")

while True:
    msg = master.recv_match(blocking=True)
    if msg:
        print(msg)
```

**2. 发送端（另一个终端运行）**

- `HEARTBEAT` [心跳协议](https://mavlink.io/en/messages/common.html#HEARTBEAT)

```python
# send.py
from pymavlink import mavutil
import time

# 连接到接收端
master = mavutil.mavlink_connection('udpout:127.0.0.1:14550')

i = 0
while True:
    master.mav.heartbeat_send(
        type=6,  # MAV_TYPE 飞行器类型
                 # 6 = 操作员控制单元/地面控制站
        autopilot=8,  # MAV_AUTOPILOT: 微型飞行器/自动驾驶仪类别（飞控）。这用于识别具体型号。
                      # 8 = 没有有效的自动驾驶仪，例如地面控制站或其他 MAVLink 组件。
        base_mode=0,  # MAV_MODE_FLAG: （位掩码）这些标志对 MAV 模式进行编码，有关有用的组合，请参阅 MAV_MODE 枚举。
                      # 0 = 全关闭
        custom_mode=0, # 由飞控决定，各系统不同。
        system_status=4  # MAV_STATE 系统状态标志
                         # 4 = 系统已启动，可能已经升空。发动机已启动。
    )
    print(f"Sent HEARTBEAT : {i}")
    i += 1
    time.sleep(1)
```

3. 运行

```sh
# 终端 1
python receive.py

# Waiting for messages...
# HEARTBEAT {type : 6, autopilot : 8, base_mode : 0, custom_mode : 0, system_status : 4, mavlink_version : 3}
# HEARTBEAT {type : 6, autopilot : 8, base_mode : 0, custom_mode : 0, system_status : 4, mavlink_version : 3}
# HEARTBEAT {type : 6, autopilot : 8, base_mode : 0, custom_mode : 0, system_status : 4, mavlink_version : 3}
```

```sh
# 终端 2
python send.py

# Sent HEARTBEAT : 0
# Sent HEARTBEAT : 1
# Sent HEARTBEAT : 2
```
