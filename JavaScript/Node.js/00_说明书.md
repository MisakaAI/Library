# Node.js 说明书

基于 Google 的 V8 引擎，把 JavaScript 用在服务端。

## 下载

[下载地址](https://nodejs.org/zh-cn/download/)

## 版本说明

LTS：长期维护版
Current：最新尝鲜版

## 安装

在终端中输入`node -v`能显示出版本号就安装成功了。
