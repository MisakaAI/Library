# 安装

```bash
# Python3
apt install python3 python3-pip

# Django
pip install django
```

## 查看安装的 Django 版本

```bash
python -m django --version
```

### 其他可能会用到的

```bash
# Apache
apt install apache2

# Nginx
apt install nginx

# SQLite
apt install sqlite3

# PostgreSQL
apt install postgresql
pip install "psycopg[binary]"
```
