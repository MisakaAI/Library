# OPC UA

OPC UA（Unified Architecture，统一架构）是工业4.0 中一个重要的通信协议（IEC 62541）
OPC UA 基于OPC基金会提供的新一代技术，提供安全，可靠和独立于厂商的，实现原始数据和预处理的信息从制造层级到生产计划或ERP层级的传输。
