# 大语言模型

## 一、什么是大语言模型

想象一个情况：

📚 你读了 **几亿本书、网页、对话记录**。
慢慢地，你会发现：

* “你好”后面常跟“有什么可以帮你？”
* “苹果是一种___” 后面可能是 “水果”
* “1+1=” 后面是 “2”

于是你学会了一个能力：

👉 **根据前面的字，猜后面的字**

这就是大语言模型最核心的能力。

### 一个简单例子

假设输入：

```text
今天的天气很好，我想去
```

模型会计算：

| 词  | 概率  |
| -- | --- |
| 公园 | 35% |
| 北京 | 20% |
| 散步 | 15% |
| 商场 | 10% |

然后选概率最大的：

```text
今天的天气很好，我想去公园
```

这就是大语言模型在做的事情：

**不断预测下一个词。**

## 二、大语言模型是怎么训练的

训练过程其实分 **3个阶段**。

### 1 预训练（Pretraining）

这是最重要的一步。

模型会看 **海量文本**：

* 书
* 维基百科
* 网站
* 论文
* 代码
* 对话

数据规模通常：

```text
1万亿 ~ 10万亿 token
```

训练目标只有一个：

```text
预测下一个词
```

例如训练数据：

```text
输入：猫喜欢喝
目标：牛奶
```

模型不断猜：

```text
猫 喜 欢 喝 ?
```

如果猜错：

```text
猫喜欢喝咖啡 ❌
```

就会被惩罚。

慢慢地模型就学会：

* 语法
* 知识
* 推理
* 编程

### 2 指令微调（SFT）

预训练后的模型其实不会聊天。

它只是补全文本。

所以要教它：

```text
用户：写一首诗
AI：好的
```

训练数据变成：

```text
User: 写一个Python排序
Assistant: ...
```

这种数据叫：

```text
instruction dataset
```

常见格式：

```text
{
 "instruction": "...",
 "input": "...",
 "output": "..."
}
```

这一步叫：

```text
SFT (Supervised Fine-Tuning)
```

### 3 人类反馈强化学习（RLHF）

最后一步让模型：

* 更安全
* 更符合人类喜好

方法：

1 人类给答案评分
2 训练一个奖励模型
3 用强化学习优化模型

流程：

```text
模型回答
↓
人类评分
↓
奖励模型
↓
强化学习优化
```

## 三、LLM结构

大语言模型本质是：

```text
Transformer
```

结构：

```text
Token
 ↓
Embedding
 ↓
Transformer Layer × 30~100
 ↓
Linear
 ↓
Next Token
```

Transformer核心是：

```text
Self-Attention
```

意思：

每个词都能看到所有词。

例如：

```text
我 今天 去 北京 旅游
```

"北京" 会影响 "旅游" 的理解。
