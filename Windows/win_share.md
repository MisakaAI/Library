# Windows 共享文件

## 登录

### 微软账号

```PowerShell
# 用户名
MicrosoftAccount\<你的微软账号(电子邮箱)>

# 密码
微软账号密码
```

### 本地用户

```PowerShell
# 查看当前用户
whoami.exe
```

## 其他注意事项

共享电脑在同一个工作组内，通常为`WORKGROUP`。
查看：设置 - 系统 - 关于 - 高级系统设置 - 计算机名 - 工作组
