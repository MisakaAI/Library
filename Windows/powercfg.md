# 电源方案

```sh
# 查看电源方案
powercfg -list

# 查看 卓越性能 GUID
powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

# 设置电源方案
powercfg -setactive <GUID>
```
